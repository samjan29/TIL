### HTML과 CSS의 개념
> HTML은 뼈대, CSS는 꾸미기!
+ HTML은 구역과 텍스트를 나타내는 코드
+ CSS는 꾸며주는 코드

### HTML 기초
> HTML은 크게 head와 body로 구성
+ head안에는 페이지의 속성 정보 (meta, script, link, title 등)
  + 페이지의 속성을 정의하거나, 필요한 스크립트를 부른다.
+ body안에는 페이지의 내용

### CSS 기초
+ HTML 부모-자식 구조에 따라 부모 태그에 적용한 스타일이 자식까지 적용되는 경우가 많다.

### 주석
+ 언제 사용?
  1. 코드를 임시로 작동하지 못하게 할 때
  2. 코드를 간단하게 설명할 때
+ 브라우저/컴퓨터가 읽지 않는다.
+ 개발자 본인이나 동료를 위해 사용

### Bootstrap
+ 미리 작성된 예쁜 CSS

### Javascript 맛보기
+ Javascript란?
  + 프로그래밍 언어로 브라우저가 알아들을 수 표준어
+ 변수
  + 사칙연산, 문자열 합치기가 가능
  + 변수명은 회사 규칙에 따라 snake case나 camel case를 사용하고 다른 특수문자나 띄어쓰기는 사용할 수 없다.
+ 리스트
  + 순서를 지켜서 가지고 있는 형태
+ 딕셔너리
  + 키(key)-밸류(value) 값의 묶음
+ 리스트와 딕셔너리는 조합해서 쓸 수 있다.
